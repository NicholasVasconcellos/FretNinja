# Task 011: Settings Screen

## Context
Task 003 created the settings store with defaults. This task builds the settings UI with all the toggles, pickers, and sliders defined in the plan. All changes persist immediately via the settings store.

## Steps
1. Build out `app/(tabs)/settings.tsx` with the following controls:
   - **Round length:** Segmented control or picker for 5 / 10 / 15 / 20 questions.
   - **Fret range:** A range selector for min/max fret (0–24). Could be two pickers or a slider.
   - **String labeling:** Toggle between numbers (1–6) and names (E, B, G, D, A, E).
   - **Show fretboard after answer:** Toggle switch.
   - **Timer per question:** Picker for Off / 5s / 10s / 15s / 30s.
   - **Smart weighting:** Toggle switch with brief explanation text.
2. Wire each control to the settings store so changes persist immediately.
3. Group settings logically with section headers (e.g., "Quiz", "Display", "Advanced").
4. Style with the dark/neon theme — use consistent control styling throughout.

## Acceptance Criteria
- [ ] All six settings from the plan are present and functional
- [ ] Changes persist across app restarts
- [ ] Controls are grouped logically with clear labels
- [ ] Smart weighting toggle includes a brief description of what it does
- [ ] Screen follows the dark + neon theme

## Files to Touch
- app/(tabs)/settings.tsx
