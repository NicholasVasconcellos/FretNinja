# Task 014: Scaffold Expo Native Module

## Context
Task 013 audited the existing pitch detection. This task creates the `modules/pitch-detector/` native module skeleton using expo-modules-api. The existing app has New Architecture enabled. This module will eventually replace `react-native-pitch-detector`, but both coexist until task 021.

## Steps
1. Create the `modules/pitch-detector/` directory structure:
   ```
   modules/pitch-detector/
     src/              # TypeScript API
     ios/              # Swift + ObjC++ bridge
     android/
       src/main/java/  # Kotlin TurboModule
       src/main/cpp/   # Oboe + JNI
     cpp/              # Shared C++ DSP
     plugin/
       src/            # Expo config plugin
   ```
2. Set up `expo-module.config.json` with iOS (Swift) and Android (Kotlin) platform config.
3. Create the config plugin in `plugin/src/index.ts` that:
   - iOS: adds microphone usage description to Info.plist, enables C++ support in Podfile.
   - Android: adds RECORD_AUDIO permission to AndroidManifest, configures CMakeLists.txt for C++ compilation with C++17.
4. Create a minimal TurboModule spec (`src/PitchDetectorModule.ts`) with stub methods: `start()`, `stop()`, `getLatestPitch()`.
5. Create the native module implementation stubs — Swift file for iOS, Kotlin file for Android — that register the module and return dummy values.
6. Wire the config plugin into `app.json` — add alongside the existing plugins. Do NOT remove `./plugins/withMicrophonePermission` (that stays until task 021).
7. Verify the module loads without crashing on both platforms via `npx expo prebuild` and a build check.

## Acceptance Criteria
- [ ] `modules/pitch-detector/` exists with full directory structure
- [ ] `npx expo prebuild` succeeds for both iOS and Android
- [ ] Calling `PitchDetectorModule.getLatestPitch()` from JS returns a dummy result without errors
- [ ] Microphone permission is declared in the new config plugin
- [ ] CMakeLists.txt is set up for Android C++ compilation (C++17)
- [ ] The existing app still works (old pitch detection is untouched)
