# Task 008: Fretboard Diagram Component

## Context
The quiz screen (Task 007) shows an optional fretboard diagram after each answer, highlighting where the correct note lives on the prompted string. This is a standalone visual component that also has potential reuse on the stats screen.

## Steps
1. Build `components/FretboardDiagram.tsx`: a visual representation of the guitar fretboard (or a relevant portion based on the fret range setting).
2. The diagram should show strings (6 horizontal lines) and frets (vertical lines), with fret numbers labeled.
3. Accept props for: which note+string combo to highlight, and optionally which fret range to display.
4. Highlight the correct fret position(s) for the given note on the given string with a neon accent color.
5. Integrate into the quiz screen: after each answer (correct or wrong), briefly show the diagram with the correct position highlighted. Respect the `showFretboardAfterAnswer` setting toggle.
6. Use the fret-to-note mapping from `utils/notes.ts` to determine which fret to highlight.

## Acceptance Criteria
- [ ] Fretboard diagram renders with correct number of strings and frets
- [ ] Correct note position is highlighted with a visible accent
- [ ] Diagram respects the fret range setting for display bounds
- [ ] Diagram shows/hides based on `showFretboardAfterAnswer` setting
- [ ] Component is reusable with configurable highlight props
