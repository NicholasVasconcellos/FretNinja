# Task 006: Home Screen

## Context
Task 001 created a placeholder home screen. This task builds it out into the app's landing page with branding, a start button, quick stats, and a round length picker. It reads from the mastery and settings stores (Task 003).

## Steps
1. Build out `app/(tabs)/index.tsx` with the FretNinja branding area (logo placeholder, tagline "Slice through the fretboard").
2. Add a prominent "Start Training" button that navigates to the quiz screen. Style it with the neon green accent.
3. Display quick stats: total rounds played and overall accuracy percentage, pulled from the mastery store.
4. Add the `RoundLengthPicker` component (`components/RoundLengthPicker.tsx`): a selector for 5 / 10 / 15 / 20 questions that updates the settings store.
5. Apply the dark theme with neon accents throughout. The home screen should feel energetic and gaming-inspired.

## Acceptance Criteria
- [ ] Home screen displays branding and tagline
- [ ] "Start Training" button navigates to quiz screen
- [ ] Quick stats show total rounds and accuracy (or "No rounds yet" for new users)
- [ ] Round length picker works and persists selection
- [ ] Screen follows the dark + neon theme
