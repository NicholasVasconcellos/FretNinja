# Task 012: Sound Effects & Visual Polish

## Context
All core functionality is complete from Tasks 001-011. This final task adds the signature FretNinja feel: ninja slice sound on correct answers, error sounds on wrong answers, and visual polish — animations, transitions, glow effects, and branding touches.

## Steps
1. Create `utils/sounds.ts`: a helper module for loading and playing sound effects using expo-av. Expose `playCorrectSound()` and `playWrongSound()` functions.
2. Source or create short SFX assets: a satisfying slice/swoosh for correct, a brief buzz/thud for wrong. Place in `assets/sounds/`.
3. Integrate sounds into the quiz flow: play correct SFX on successful note detection, wrong SFX on timeout or incorrect.
4. Add animations to key interactions:
   - NotePrompt entrance animation (slide/fade in for each new question).
   - FeedbackOverlay: scale/fade animation for the checkmark and X.
   - TimerBar smooth countdown animation.
   - "Start Training" button: subtle pulse or glow effect.
5. Add neon glow effects to key UI elements (buttons, active states) using shadow styling.
6. Add screen transition animations between quiz → results and home → quiz.
7. Place logo/branding assets in `assets/images/` and integrate into the home screen.

## Acceptance Criteria
- [ ] Correct answer plays a slice/swoosh sound
- [ ] Wrong answer / timeout plays a buzz/thud sound
- [ ] Sounds load without blocking the UI
- [ ] Key UI elements have smooth animations (prompt transitions, feedback, timer)
- [ ] Neon glow effects are applied to buttons and accent elements
- [ ] Screen transitions feel polished
- [ ] Logo/branding is visible on the home screen
