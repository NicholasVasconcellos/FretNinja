# Task 010: Stats & Mastery Dashboard

## Context
Tasks 003 and 002 set up the mastery store and scoring utilities. This task builds the stats screen with the 12×6 mastery heatmap grid, overall stats, weakest combos list, and recently improved combos. This is the user's primary way to track fretboard mastery progress.

## Steps
1. Build `components/MasteryGrid.tsx`: a 12 notes × 6 strings grid (heatmap). Each cell is color-coded by mastery level: red (new/weak) → yellow (learning) → green (mastered). Cells should show the note name and be tappable for detail (stretch).
2. Build out `app/(tabs)/stats.tsx` with:
   - Overall stats section: total rounds played, lifetime accuracy, total notes played.
   - The MasteryGrid component.
   - Weakest 5 combos list with note+string and accuracy.
   - Recently improved combos (optional for MVP — include if straightforward).
3. Pull all data from the mastery store via the `useMasteryData` hook.
4. Handle the empty state gracefully (no rounds played yet).

## Acceptance Criteria
- [ ] Mastery grid displays all 72 combos in a 12×6 layout
- [ ] Grid cells are color-coded by mastery level (red → yellow → green)
- [ ] Overall stats (rounds, accuracy, notes played) are displayed
- [ ] Weakest 5 combos are listed
- [ ] Empty state is handled (no data yet)
- [ ] Screen follows the dark + neon theme
